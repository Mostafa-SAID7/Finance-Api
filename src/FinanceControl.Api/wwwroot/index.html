Quick Start

Get the API running in 5 minutes

1

Clone Repository

git clone https://github.com/Mostafa-SAID7/Finance-Api.git
2

Start Services

cd src/FinanceControl.Api && docker-compose up -d
3

Run Application

dotnet run
4

Access API

http://localhost:5012/swagger

📖 API Endpoints

  • GET /api/balance - Get balance
  • POST /api/expenses - Create expense
  • POST /api/incomes - Create income

🔗 Services

  • API: http://localhost:5012
  • Swagger: http://localhost:5012/swagger
  • Kibana: http://localhost:5601
